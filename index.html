# Chardonneret est un nom vernaculaire ambigu en français, pouvant désigner plusieurs espèces différentes d'oiseaux. Ce nom vernaculaire est aussi à la base de plusieurs noms normalisés ou de noms vulgaires créés pour la nomenclature scientifique en français.
